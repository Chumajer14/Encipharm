Titulo: Solicitud de Comunicacion de Inicio de Actividades para Establecimientos de Productos de Alimentacion Animal | SAG

Este tramite permite que empresas privadas del rubro de alimentos para animales informen al SAG el inicio de sus actividades. La informacion se usa para controlar que la produccion y comercializacion de estos productos se ajuste a la normativa vigente.

Dirigido a:
- Fabricantes.
- Elaboradores.
- Importadores.
- Distribuidores.
- Almacenes.
- Locales de expendio de alimentos, suplementos, aditivos e ingredientes destinados a alimentacion animal.

Marco regulatorio:
- Reglamento de Alimentos para Animales, Decreto N 4/2016.
- Debe comunicarse inicio de actividades de importacion, exportacion, produccion, almacenamiento o envasado de alimentos para animales.

Requisitos operacionales citados por SAG:
- Programa de Aseguramiento de Calidad en fabricas o elaboradoras de alimentos y suplementos para animales.
- Requisitos estructurales y operacionales de fabricas de ingredientes de origen animal destinados a alimentacion animal.
- Requisitos para funcionamiento de fabricas y plantas elaboradoras de alimentos y suplementos.
- Requisitos para locales de expendio de alimentos, suplementos y aditivos.
- Requisitos para bodegas destinadas al almacenamiento de alimentos para animales.

Ante cambios de datos, cese temporal o cierre definitivo, el propietario o representante debe comunicarlo al SAG.

El tramite no tiene costo y su vigencia es indefinida hasta que el propietario cese alguna actividad o cierre el establecimiento.
